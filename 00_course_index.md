### How to Use This Notebook
- Each chapter is a self-contained `.md` file with **theory → code → edge cases → exercises**
- Code examples compile with `gcc -Wall -Wextra -std=c11`
- Embedded examples target ARM Cortex-M (STM32) but concepts apply universally
- Edge cases are marked with ⚠️, tips with 💡, and embedded notes with 🔧

### Prerequisites
- Basic C syntax (variables, functions, loops, pointers)
- A C compiler (GCC, Clang, or MSVC)
